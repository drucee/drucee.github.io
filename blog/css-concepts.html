CSS Concepts: Margin, Border, Padding

February 28, 2016

When writing CSS for an HTML page, it is important to remember that every element on the page takes the form of a rectangle, or box. It doesn't matter what kind of element you're working with, whether it is a div, a span, a paragraph, an image, a link, or anything else--everything in HTML is a box. In fact, one of the important first steps when translating a webpage from rough sketch or wireframe to browser-displayable code is "boxifying" the design; that is, enclosing elements in boxes and arranging them appropriately.

Examples of "boxified" wireframes. Image courtesy codepen.io.

But how do we ensure those boxes' relative placement on a page, and how do we keep them from overlapping? We can use CSS to make this simple. HTML pages are built according to the document-object (DOM) model, and every HTML document contains objects within it, in a sort of tree-like structure, whereby objects are nested in other objects. Those objects (Web content) are surrounded by three different types of buffers, from innermost to outermost: the padding, the border, and the margin.

The "box" model of an object in HTML. Image courtesy w3.org.

The outermost layer of an object's "box" is its margin. The margin is a transparent buffer that separates the object from other objects, or from the edges of the browser window. The border is a visible box that extends all the way around the object. Borders can be made any thickness, and can be solid or dashed. The innermost layer of the "box" before the content is the padding. This layer provides a buffer between the content and the border. Padding can be made either visible or invisible.

The dimensions of the padding, border, and margin rectangles default to zero, but we can use CSS to adjust them. Each layer can be adjusted wholesale (using the padding, border, and margin properties within the relevant object's CSS selector and setting a single value, or a value can be set for each individual side (top, bottom, left, right). Furthermore, borders can be made solid or dashed, as well as given a color.
